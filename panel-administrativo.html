Panel Administrativo ALOTAR

Guía práctica para operar tu sistema de reservas.

1. Acceso

  • Abre el correo recibido.
  • Haz clic en Panel Administrativo.
  • Ingresa con tu PIN si es requerido.

⚠️ No compartas este acceso con clientes.

2. Qué puedes hacer

  • Revisar reservas
  • Validar horarios
  • Controlar sedes
  • Ver actividad del sistema

3. Validar reservas

  • Confirma nombre, fecha y hora
  • Verifica la sede
  • Revisa que aparezca en el calendario

✔ Haz pruebas antes de compartir el sistema

4. Link público

  • Este es el enlace que compartes con tus clientes
  • No cambia
  • Debe ser probado antes

5. Errores comunes

  • No configurar sedes
  • No validar horarios
  • Compartir el link incorrecto
